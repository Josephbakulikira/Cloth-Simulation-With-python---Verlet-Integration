 ## Controls
        LEFT_CLICK   ->  to select a point
        RIGHT_CLICK  ->  to deselect a point
        SPACE while  ->  selecting a point remove one of if joint
        MOUSE WHEEL  ->  click while a selecting a point make it a static point or lock node
